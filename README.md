# Super Source

## Installation

##### Instructions for using Super Source

Create a new, empty directory. Inside this directory, create a file named Gemfile, and add the following to it:

```ruby
source 'https://rubygems.org'

gem 'super_source'
```

Then execute:

    $ bundle install
